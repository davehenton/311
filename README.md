The source code for the future [311.boston.gov](https://311.boston.gov).

## Developers

This is a Rails 5 application that connects to the Swiftype API and the City of Boston's Open 311 API. In order to get setup you should:

 * Checkout the code
 * Install any necessary gems with `bundle install`
 * Create a `.env` file with your environment specific configuration based on the `.env.sample`
 * Run `rake db:migrate`
 * Use foreman to run everything with `foreman start`
